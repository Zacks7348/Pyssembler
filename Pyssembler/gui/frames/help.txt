SYSCALL

What do SYSCALL instructions do?

SYSCALL instructions perform system services such as IO. 

How to use SYSCALL instructions?

Except for return registers, MIPS registers will not be updated during a system call.

You can execute a specific system call by following these steps:
1.  Load the service number in $v0
2.  Load any argument values in $a0, $a1, or $a2 as specified
3.  Execute the SYSCALL instruction
4.  Retrieve any return values from result registers as specified
